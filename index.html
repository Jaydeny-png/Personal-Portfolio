My Skills

JavaScript

JavaScript is a coding language that makes websites interactive, like showing messages, changing colors, or reacting when you click buttons.

HTML

HTML (HyperText Markup Language) is the basic code used to create and structure content on a webpage, like text, images, and links. It tells the browser what to show and how it's organized.

CSS

CSS (Cascading Style Sheets) is the code used to style HTML content. It controls how things look on a webpage—like colors, fonts, sizes, and layout.

Python

Python is a simple, easy-to-read programming language used to build websites, apps, games, and even control robots or analyze data. It’s great for beginners and very powerful.

JSON

JSON (JavaScript Object Notation) is a simple format for storing and sharing data. It uses key-value pairs and is easy for both humans and computers to read.

Bootstrap

Bootstrap is a free tool (a CSS framework) that helps you quickly design good-looking, responsive websites. It includes ready-made buttons, layouts, and styles that work well on phones, tablets, and computers.
